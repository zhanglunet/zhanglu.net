## 用途

做有辨识度、生产级、设计质量高的前端界面。

## 何时用

用户要做：

- Web 组件、页面、artifact、海报、应用
- 网站、landing page、dashboard
- React 组件、HTML / CSS 布局
- 给任何 web UI 加样式 / 美化

## 设计倾向

生成有创意、打磨过的代码和 UI——**避开通用 AI 美学**：干净但有辨识度，不像批量生成的样板。

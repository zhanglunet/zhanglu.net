## 用途

用 Chrome DevTools MCP 分析 web 性能。

## 测什么

- **Core Web Vitals**：LCP、INP、CLS
- **辅助指标**：FCP、TBT、Speed Index

## 查什么

- 渲染阻塞资源
- 网络依赖链
- 布局偏移
- 缓存问题
- 可访问性缺口

## 何时用

审计、profile、debug 或优化页面加载性能、Lighthouse 分数、站点速度。

## 知识倾向

偏向检索最新文档，而非依赖预训练知识。

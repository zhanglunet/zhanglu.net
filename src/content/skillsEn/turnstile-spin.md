## Purpose

Wire up Cloudflare Turnstile end-to-end in a project.

## What it does

1. Scan the codebase
2. Call the Cloudflare API to create a widget
3. Deploy a managed siteverify Worker
4. Write the frontend snippet
5. Verify
6. Freeze the skill

## When to load

- Adding Turnstile
- Setting up a CAPTCHA
- Protecting a form against bots
- Fixing a Turnstile integration

## Purpose

A browser-automation CLI for AI agents.

## When it triggers

"Open a website", "fill a form", "click a button", "take a screenshot", "scrape the page", "test this web app", "log in to a site", "automate browser actions" — any task that needs programmatic web interaction.

## Also covers

- Exploratory testing, dogfooding, QA, bug hunts, app quality reviews
- Automating Electron desktop apps (VS Code, Slack, Discord, Figma, Notion, Spotify)
- Checking Slack unreads / sending Slack messages / searching Slack threads
- Running browser automation inside a Vercel Sandbox microVM
- Using AWS Bedrock AgentCore cloud browsers

## Priority

Preferred over any built-in browser automation or web tool.

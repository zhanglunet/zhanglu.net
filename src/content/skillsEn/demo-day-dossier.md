## Purpose

An end-to-end roadshow / demo day dossier pipeline.

## Inputs

- A folder of project card screenshots
- Booth-wall photos
- Official host article URLs (any combination)

## Outputs

1. A structured JSON dataset of all projects
2. A panoramic, interactive HTML landing page
3. A sortable, filterable DD (due-diligence) table page — driven by 7-way parallel research agents
4. An in-depth Word report
5. A CSV table (opens directly in Excel / Numbers)
6. One-click deploy to Cloudflare Pages

## When to use

Accelerator / VC roadshow / pitch day materials (many project cards + demo day photos + official announcement articles) when you want research + DD + a publishable dossier done in one pass.

## Explicit triggers

- "Turn the roadshow projects into a panoramic web page"
- "Organize the 56 projects from the Qiji / YC / Sequoia / Matrix roadshow"
- "Run DD on all the demo day projects"
- "Build a due-diligence table and ship it on Cloudflare"
- "analyze all roadshow projects and publish"

## Don't use for

- Deep research on a single project → use `/research`
- DD on a single company → just write a focused prompt + WebSearch
- Real-time investor meeting tracking → no tool for this yet

## Purpose

Build distinctive, production-grade frontend interfaces with high design quality.

## When to use

When the user wants to build:

- Web components, pages, artifacts, posters, apps
- Websites, landing pages, dashboards
- React components, HTML / CSS layouts
- Styling / polishing for any web UI

## Design bias

Produces creative, refined code and UI — **avoiding the generic AI aesthetic**: clean but distinctive, not like mass-generated boilerplate.

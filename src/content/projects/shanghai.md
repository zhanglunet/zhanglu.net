## 是什么

一个**非官方**的会议回顾站，把 2026 AI 愿景论坛（上海站）的 30 场分享，整理成一份能查、能逛、能顺藤摸瓜的知识资产：

- **知识星图**：99 个实体（人物 / 机构 / 概念 / 事件）、288 条共现关联的交互式知识图谱，支持搜索、筛选、拖拽，点节点直接进词条。
- **智能纪要**：每场分享含金句、信息图、模型解读与要点点评。
- **知识库**：100+ 实体页面（公司 / 人物 / 概念 / 事件），带出现频次与双向互链。
- **资源库**：131 张现场幻灯片（人脸已打码）。

纯静态 HTML / CSS / JS，无构建依赖、无第三方库，部署在 Cloudflare Pages 自动上线。

## 为什么做

会议现场信息密度极高，但散落在几十场分享、几百张幻灯片和当场的记忆里，一周后基本"听完就忘"。把纪要、实体、关联结构化进一个可检索、可互链、可缩放的站点，就能从"听完就忘"变成"事后随时回来查、顺着关系图接着挖"。

AI 内容由**飞书妙记 + 大模型**自动生成，全站透明标注"仅供学习参考"。

## 你能怎么用

- 想快速回顾某场分享 → 智能纪要页看金句 + 要点点评。
- 想顺着一个人 / 一家公司 / 一个概念挖下去 → 知识星图或知识库词条，跟着双向链接走。
- 想拿原始素材 → 资源库翻现场幻灯片。

适合：没到现场但想补课的从业者、做赛道 mapping 的研究员、想写行业速记的媒体。

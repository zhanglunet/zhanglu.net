## 是什么

一套基于 Claude Skill + 多评委合议的品牌判断流水线。给一个品牌，自动跑完 5 位评委（傅盛 / Steve Jobs / 李可佳 / 吴俊东 / 张一鸣 等）的独立打分、异议热力图、影响力构造图，输出版本化的 Markdown + HTML 报告。

## 为什么做

品牌影响力的判断长期靠"感觉"。把判断分解成可追溯的 5 维度（reasoning / evidence / counter / falsifiability / real-world resilience）+ 不同 mental model 的评委合议后，每一条结论都能 attribute 到证据。

## 你能怎么用

零依赖入门：装完直接 `/mba <brand> --quick --no-judges` 跑一份单视角速读。满意了再加 panel 和云浏览器。

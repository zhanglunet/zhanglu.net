## 是什么

把路演现场的项目卡片照片、展区墙照片、官方文章，一次性变成：
1. 结构化 JSON 数据集
2. 全景式交互 HTML landing page
3. 可排序/筛选的 DD 表（7 路并行调研 agent 驱动）
4. Word 深度报告
5. 可在 Excel/Numbers 打开的 CSV
6. 一键部署到 Cloudflare Pages

## 为什么做

路演现场信息密度高、但散落在照片和公众号文章里。把它们抽成可索引、可对比、可发布的结构化资产，比"看完就忘"高出一个量级。

## 你能怎么用

把同类型的 demo day / 加速器 / VC 路演现场素材丢进 `/demo-day-dossier`，半小时跑完全套。

# 本周做了什么 · 2026-W29（07-13 → 07-19）

**一句话**：这周把三个「决策智能体」产品各往前推了一个大版本（**MBA v0.6 / Boss v1.18 / OAF v0.43**），同时在个人站上把它们背后的共同想法——「**为 AI 原生组织的 C-suite，做一套可调研、可打分、可复盘的决策智能体**」——讲清楚、摆到了一起。

## 三个产品，一周三个大版本

### 🎯 MBA · 给 CMO 的品牌影响力审计（mbabrand.com）— v0.6

- **从"品牌监控"长成"关系宇宙"**：品牌 + 创始人 + 产业 + 组合。一周纳入 NVIDIA / Apple / Google / Microsoft / Amazon / Huawei / DeepSeek / Tesla / Palantir（各含完整审计报告 + 创始人档案）。
- **创始人晚餐**：把两位创始人放一桌，按 5 个镜头推演合作（如梁文锋 × 黄仁勋「芯片与模型」；AI 演绎、非真实合作）。
- **舆情监控做成全自动闭环**：每天自动发现候选事件 → 大模型预分类 → 一键采纳入库 → 触发品牌重审。
- 配套：全维度知识星图、舆情驾驶舱、飞书分层预警。

### 🧭 Boss · 给 CEO 的决策智能体 → 战略 OS（bossagent.cc）— v1.18

- **从"评审一个议题"升级为「战略 OS」**：不只是给一次判断打分，而是把判断连成一条**组织回路**——决策捕获 → 行动台账 → 到期跟催（T+7 / T+30）→ 复盘沉淀。判断力第一次变成可积累、可回看的资产。
- **评委体系对外开放**：公开「评委全景」页，每位评委的方法论（doctrine）可查；战略 OS 方法论专题页上线。
- **公开判例库**：访客提交的议题与判断报告公开保留一段时间，供后来者浏览。

### 📊 OAF · 给 CFO 的 AI 投研工作台（oaf.world）— v0.43（一周约 14 个版本）

- **看板从只读升级为"可探索驾驶舱"**：全局切片、跨面板联动、下钻、深链、⌘K 搜索、自选关注、对比模式。
- **对话调研接真**：随口一问，大模型只做意图路由，取数交给确定性工具——绝不编数字。
- **卫星互联网专题群**：SpaceX / 垣信·千帆 / 中国星网 / 星座竞赛四张深度看板 + 行业异动实时预警。
- **首页实时化**：接实时后端 + 实时 K 线行情（美股 / A 股 / 卫星互联网，"上市以来"长序列）。

## 一条主线：为 AI 原生组织而设计

三个产品面向不同职能（CMO / CEO / CFO），底层却是同一套骨架：**多评委独立合议 → 可打分可证伪 → 版本化 + 归因校验 → 反捏造**。当一个组织把越来越多的执行交给 agent，顶层判断也该长在同一套可追溯的底座上，而不是继续靠"感觉"。

MBA 把品牌判断自动化，Boss 把决策沉淀成组织回路，OAF 把投研判断实时化——**三个席位，一套骨架**。

## 顺手：把这条主线讲清楚了（zhanglu.net）

- 新开 **C-suite 专题**（含 logo「三弧一点，一枚印」、品牌页、一篇设计思想长文），把三件套的共同哲学摆到一起。
- 给每个项目配了**网站截图 + 源码行数 + 面向人群**标注；还翻出了 20 多年前的**往期作品**——1999–2007 做的几个中文个人网站，整体重建成静态站永久保存。

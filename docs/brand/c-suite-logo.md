# C-suite 决策智能体 — Logo 设计文档

> 资产在 `public/brand/`，线上品牌页 https://zhanglu.net/c-suite/brand ，专题页 https://zhanglu.net/c-suite 。

## 一句话

**三弧一点，一枚印。**

## 概念推导

给「C-suite 决策智能体」做标识，要回答三个问题：它管什么（决策）、它由什么构成（三个席位）、它信什么（人最终拍板）。三个答案各对应一个图形元素：

### 1. 一枚印章 —— 决策的终极动作

站点的 accent 色 `#b14b3a` 是**朱砂**——传统印泥的颜色。而「盖章」正是中文语境里决策的终极动作：拍板、用印、生效。

这套产品的核心主张之一是**版本化冻结**——每份判断书生成不可变快照（Boss 里是 `chmod 444`），供事后验真。印章天然携带这层意思：**盖下去，就是一个不可篡改的版本**。

于是外框取印章形：圆角方（120 × 120，圆角 24），朱砂色，线宽 7。阳刻（描边）为主形，阴刻（满铺反白）为反白形，正好对应篆刻的两种刻法。

### 2. 三段弧合成一个 C —— 三个席位，独立而合议

C-suite 的 C，由**三段断开的弧**构成：

- 三段弧 = 三个席位：Boss（CEO）、MBA Brand（CMO）、OAF（CFO）
- 弧与弧**断开** = 评委独立打分、互不相见（合议前不通气，是这套方法论的硬规则）
- 三段合起来**读作一个 C** = 合议成一体，一套骨架

几何：圆心 (60, 60)，半径 30，线宽 9，圆头端点。C 的开口朝右、张角 80°（呼应「开放的判断，不闭环自证」）；三段弧各 85.3°，段间隙 12°。

### 3. 中心一点墨 —— 锚点，人在回路

圆心处一个**墨色**（`#1a1a1a`，非朱砂）实点，半径 6.5：

- 它是 **锚点**（Boss 里的 anchor）——最终按下按钮的那个人
- 全图唯一的墨色元素：机器（朱砂流水线）围绕着人（墨点）转，**不是取代人拍板**
- 视觉上它也让 C 有了「视点」，整个 mark 隐约像一只眼——审视、复盘

## 字体

- 组合标识中文 / 拉丁：衬线（Georgia / Source Serif Pro / Noto Serif SC 栈，与站点标题一致）
- 副标语：无衬线小字 + 字距拉开

## 使用规范

- **最小尺寸**：mark 短边 ≥ 20px（心点仍可辨）；lockup 宽 ≥ 180px
- **留白**：四周至少留 mark 宽度的 1/6（约一个心点直径）
- **别做**：不改颜色（只允许朱砂 / 墨 / 反白三态）、不拆开三段弧单独用、不去掉心点、不加投影渐变、不斜置
- 深色背景一律用阴刻版（solid）

## 设计取舍记录

- 考虑过「三把椅子」「圆桌俯视」等具象方案，弃：太图解，缩小即糊
- 考虑过纯字标（C-SUITE），弃：没有承载「合议 / 版本化 / 人在回路」任何一层语义
- 三段弧的间隙曾试 20°，缩小后 C 断裂感过强，收敛到 12°
- 心点曾试朱砂色，与弧同色后「人」的层次消失，定为全图唯一墨色元素
